Alumnos

En la parte superior de la página encontrará una opción llamada Alumnos que desplegará una lista de opciones: Añadir, Modificar, Eliminar y Ver Listas.

  • Añadir

    Esta es la pestaña para agregar un nuevo alumno al jardín.

  • Modificar

    Permite editar la información de un alumno ya registrado.

  • Eliminar

    Opción para remover alumnos del sistema.

  • Ver Listas

    Muestra un listado completo de los alumnos registrados.

Responsables

En la parte superior de la página encontrará una opción llamada Responsables (maestras, preceptoras, etc.) que desplegará una lista de opciones: Añadir, Modificar, Eliminar y Ver Listas.

  • Añadir

    Esta es la pestaña para agregar una nueva persona responsable al jardín.

  • Modificar

    Permite editar la información de un responsable ya registrado.

  • Eliminar

    Opción para remover responsables del sistema.

  • Ver Listas

    Muestra un listado completo de los responsables registrados.

Registro

En la parte superior de la página encontrará una opción llamada Registro que desplegará una lista de opciones: Añadir, Modificar, Eliminar y Ver Listas.

  • Añadir

    Esta es la pestaña para agregar un nuevo registro de asistencia.

  • Modificar

    Permite editar registros existentes en el sistema.

  • Eliminar

    Opción para remover un registro si fue creado por error.

  • Ver Listas

    Muestra todos los registros guardados en el sistema.
